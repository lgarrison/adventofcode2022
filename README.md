## Usage
To make a new day:
1) `$ cd src; cp -r template dayX`
1) Put the input in `src/dayX/input.txt`
1) Add a new `[[bin]]` section to `Cargo.toml`

To run:
: `$ cargo run --bin dayX`

To run fast:
: `$ cargo run --release --bin dayX`
